ข้อมูลหลักสูตร

ชื่อหลักสูตร : วิศวกรรมศาสตรบัณฑิต

รหัสหลักสูตร : 25620151100293

ระดับการศึกษา : ปริญญาตรี ชั้นปีที่ 1

มหาวิทยาลัย : เทคโนโลยีพระจอมเกล้าพระนครเหนือ

คณะ : เทคโนโลยีและการจัดการอุตสาหกรรม

หลักสูตรนี้ออกแบบมาเพื่อผลิตวิศวกรที่มีความรู้ ความสามารถ และทักษะเชิงลึกในด้านเทคโนโลยีสารสนเทศ ระบบเครือข่ายคอมพิวเตอร์ การสื่อสารข้อมูล และระบบความมั่นคงปลอดภัยของข้อมูล โดยมุ่งเน้นให้ผู้เรียนสามารถออกแบบ พัฒนา ติดตั้ง วิเคราะห์ และบริหารจัดการระบบเครือข่ายและระบบสารสนเทศ ในระดับองค์กรได้อย่างมีประสิทธิภาพ รวมถึงสามารถประยุกต์ใช้เทคโนโลยีใหม่ๆ เพื่อตอบสนองต่อการเปลี่ยนแปลงของโลกดิจิทัลและความต้องการของภาคอุตสาหกรรมได้อย่างยั่งยืน
